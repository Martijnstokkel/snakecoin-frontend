Deze website heb ik gemaakt met de spring boot maven framework voor mijn back-end. Voor mijn front-end heb ik gebruik gemaakt van angular. Hiervoor heb ik gekozen omdat het maken van een link tussen front-end en back-end makkelijk word gemaakt met deze frameworks. Ik heb hierdoor meer kunnen focussen op de invulling van de website en de functionaliteit van deze invulling. Ik wil bij eventuele volgende projecten kijken naar andere frameworks en hoe ik hier mee om kan gaan.
